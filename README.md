## 📖 Sobre

Este repositório foi criado para **armazenar meu progresso e aprendizados** durante o curso de **NestJS** da **Rocketseat**.

O projeto serve como ambiente de estudo para aplicação de conceitos como:

- Clean Architecture
- Domain-Driven Design (DDD)
- SOLID
- Testes automatizados
- APIs REST
- Autenticação com JWT
- Upload de arquivos
- Integração com PostgreSQL

> **Este projeto possui finalidade exclusivamente educacional e não representa um produto final.**

---

## 🚀 Tecnologias

- NestJS
- TypeScript
- Prisma ORM
- PostgreSQL
- Docker
- Vitest
